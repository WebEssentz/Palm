# Layout Component Vocabulary

## Section Types → HTML Patterns
nav        → sticky header with logo + links + CTA
hero       → full-width centered headline + subtext + CTA buttons
input      → centered textarea/input with toolbar row inside one box
cards      → grid of 2-4 equal cards with icon/image + heading + text
sidebar    → flex row: fixed-width left panel + scrollable main content
pricing    → grid of 2-3 pricing tiers with feature lists
table      → full-width data table with thead + tbody
form       → stacked labeled inputs with submit button
banner     → full-width colored strip with text + button
footer     → multi-column links + copyright row

## Layout Structures
centered   → max-width container, margin auto, items-center
sidebar    → display:flex, aside fixed width, main flex:1
dashboard  → CSS grid with named areas: header, sidebar, main, widgets
landing    → stacked sections, alternating left/right content
